# chardet Project Reading Notes

## 1. Project Type

chardet is a Python library, not a simple standalone app.

## 2. Main Purpose

It detects the character encoding of text data, such as UTF-8, ASCII, GB2312, ISO-8859-1, and other encodings.

## 3. What I Can Learn From It

- How a professional Python package organizes source code
- Why many Python projects use a `src/` folder
- How tests and documentation are separated from source code
- How README explains usage with short examples

## 4. Project Structure

- `src/chardet/`: main source code of the library
- `tests/`: test files
- `docs/`: documentation
- `scripts/`: helper scripts
- `.github/workflows/`: GitHub automation
- `README.md`: project introduction and usage guide

## 5. What Seems Difficult

- The encoding detection algorithm
- The large number of files
- The professional packaging structure
- The test and documentation system

## 6. My Current Goal

I do not need to fully understand them now. I only want to observe the structure and learn how a mature Python library is organized.
